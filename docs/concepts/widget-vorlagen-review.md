# Review-Worksheet — Widget+Vorlagen-Foundation

**Begleitfile zu** `widget-vorlagen-foundation.md`. Eine Tabelle pro Hauptabschnitt. User kommentiert die zwei rechten Spalten, setzt Status auf `bestaetigt` / `geaendert` / `offen`.

**Workflow:**

1. User liest Konzept-Hauptfile Abschnitt fuer Abschnitt.
2. Pro Item hier Status setzen + Kommentar wenn `geaendert` / `offen`.
3. Wenn alle auf `bestaetigt` oder `geaendert (mit Folge-Vermerk)`: Implementierungs-Plan-File schreiben.

**CSV-Variante:** identisch in `widget-vorlagen-review.csv` fuer Excel-Editing.
